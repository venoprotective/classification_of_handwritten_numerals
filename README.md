# Классификация рукописных цифр MNIST

Нейронная сеть для распознавания рукописных цифр (0-9) на numpy.

## Требования

pip install numpy pandas matplotlib scikit-learn

## Быстрый старт

### Запуск обучения и графиков

python main.py

После запуска в консоли вы увидите процесс обучения с точностью на каждой эпохе и финальную точность на тестовых данных. Также откроются два графика: график точности (обучение и валидация) и график ошибки MSE.

### Запуск только визуализации

python plot.py

## Ожидаемые результаты

Точность на тестовых данных должна быть около 95-97%.

Пример вывода в консоль:

Эпоха: 001/050 Train MSE: 0.0892 Train Acc: 94.56% Valid Acc: 94.12%\n
Эпоха: 002/050 Train MSE: 0.0751 Train Acc: 95.23% Valid Acc: 94.78%
...
Accuracy in testing: 95.67%

## Настройка параметров

В файле main.py можно изменить параметры модели:

num_hidden = 128 - количество нейронов в скрытом слое
num_epochs = 50 - количество эпох обучения
learning_rate = 0.1 - скорость обучения
minibatch_size = 100 - размер мини-батча
                                                                                                                                                                                                                                                                
Если не хватает памяти, уменьшите minibatch_size.
